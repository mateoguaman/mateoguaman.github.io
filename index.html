S.A.M.: Semantic Autonomous Mapping

We created an autonomous semantic mapping functionality for a Turtlebot robot using ROS. It autonomously navigates the building given a set of waypoints. In real time, it detects the door signs in Halligan Hall and it runs optical character recognition using Google Cloud Vision to build a semantic representation of the building.
